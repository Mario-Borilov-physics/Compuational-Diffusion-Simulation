# The Project
This project is a computational simulation that simulates the physical process of diffusion in Python.
# Features
- physics based simulation
- the calculations are done via NumPy
- the simulation is animated via FuncAnimation from matplotlib.animation
- visualisation is done by matplotlib.pyplot using pcolormesh() to visualise the scalar field of the concentration
# How to run
``` bash
pip install numpy matplotlib
run python diffusion_sim.py
